## Problem Statement
Most of the educational platforms in Nepal are facing problems in managing their students, teachers, courses, exams, assignment and other administrative tasks which they are currently doing manually.

Managing and tracking is a tedious and time consuming process

# What is dManage.com?
dManage.com is a web application that helps educational platforms to manage their students, teachers, courses, exams, assignment and other administrative tasks. 

It provides a complete platform for the educational institutions to manage your entire operations by just few clicks. And boom! You are done everything is secured and tracked by dManage teams

Key highlights of the dManage.com project include:

1. User Friendly Interface
2. Easy to Use
3. Secure
4. Affordable Price
5. Reliable
6. Fast
7. Accessible
8. Maintainable

# Tech Stack
- Language: Typoescript
- Frontend: Next, React, ReduxToolkit
- Backend: Node, Express
- ORM: Prisma
- Database: PostgreSQL
